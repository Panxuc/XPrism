# XPrism

[English](README.en.md) | [简体中文](README.md) | **繁體中文**

## 關於

![appicon](.readme/appicon.png)

清華大學電子工程系硬體設計大賽參賽作品。

XPrism 是基於 ESP32-PICO-D4 開發的智慧眼鏡。

此倉庫同時包含基於 .NET MAUI 開發的 Android 端控制器。
